Triangle Trigonometric Functions

2024-04-03 :: 1 min read (37 words)
#maths  #trigonometry  #wave-functions 

Sine⌗

Right Triangles⌗

  • sin(θ) = opposite / hypotenuse

Non-Right Triangle⌗

  • a / sin(A) = b / sin(B) = c / sin(C)

Cosine⌗

Right Triangle⌗

  • cos(θ) = adjacent / hypotenuse

Tangent⌗

Right Triangle⌗

  • tan(θ) = opposite / adjacent
